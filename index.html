About

品質保証（QA）業務に従事する傍ら、フロントエンドエンジニアへのキャリアチェンジを目指し、HTML / CSS / JavaScript の学習を継続しています。

これまでに通信営業やダンススタジオ運営に携わり、顧客対応力や課題解決力を培ってきました。また、SNS運用やデザイン制作、イベント企画など幅広い業務を担当してきた経験があります。

動画制作においては、知人の案件をお手伝いする形で実務にも関わり、企画から編集まで一貫して対応してきました。

現在は、これまでの動画編集やデザインの経験と掛け合わせ、「ユーザーにとって分かりやすく魅力が伝わるフロントエンド開発」を強みとしてスキル向上に取り組んでいます。
